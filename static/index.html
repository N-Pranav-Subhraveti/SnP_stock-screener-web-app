Adaptive Trend Filter

Price vs 20-Day EMA: Minimum percentage of current price relative to the 20-day Exponential Moving Average. Value of 100 means price must be above EMA20. Lower values (95-99) allow price to be slightly below EMA20 while still considering uptrend.
Price vs 50-Day EMA: Minimum percentage of current price relative to the 50-day Exponential Moving Average. Value of 100 means price must be above EMA50. Lower values (90-98) allow price to be further below EMA50 while maintaining trend strength assessment.
RSI Minimum: Lowest acceptable RSI value. Prevents selecting stocks that are extremely oversold (RSI < 20-30) which might indicate strong downtrend. Higher values (25-40) avoid deeply oversold conditions.
RSI Maximum: Highest acceptable RSI value. Prevents selecting stocks that are overbought (RSI > 70-80). Lower values (65-75) ensure stock has room to grow without being overextended.
Volume Ratio Minimum: Minimum ratio of current volume compared to 20-day average volume. Values above 1.0 indicate above-average volume (strong interest). Values 0.7-1.0 allow normal to slightly below-average volume while still confirming trend.
Price vs 20-Day High: Percentage of current price relative to the 20-day high. Value of 100 means price is at 20-day highs. Lower values (75-95) allow stock to be further from recent highs while still in uptrend. Helps identify stocks with potential breakout opportunities.
Required Conditions: Number of trend conditions that must be met for a stock to pass the filter. Higher values (3-5) make screening stricter. Lower values (1-2) make it more lenient. Each strategy has different total available conditions (MA Crossover: 5, RSI: 4, Supertrend/Heikin-Ashi: 3).
